# 🤖 Discord.js v14 Bot Template

## 🚀 Setup
⚡ ```npm install discord.js mongoose```
🔥 ```node index.js```

## ✨ Features
👑 Developer commands
📊 MongoDB integration
🔄 Ping system
⚠️ Error handling

## 📌 Requirements
💻 Node.js v16.9.0+
🗄️ MongoDB database
🔑 Discord Bot Token
